# EduPilot AI – Agentic School ERP Assistant - API Reference

This document describes the backend REST API endpoints, request schemas, headers, response payloads, and example commands.

---

## 📥 `POST /chat` - Submit User Message

Process a natural language user query for a given student ID. The agent will parse intents, select tools, query the ERP mock database, execute reasoning, and log metrics.

### Request Details
- **URL Path:** `/chat`
- **Method:** `POST`
- **Headers:**
  - `Content-Type: application/json`
  - `X-Groq-API-Key` *(string, optional)*: Injects a custom Groq API Key to enable Llama 3.3. If omitted, the server operates on default configurations or offline mock fallback.

### Request Body Schema
```json
{
  "student_id": "string (Required, e.g. ST101)",
  "message": "string (Required, e.g. Show my midterm marks and check pending fees.)"
}
```

### Response Schema (200 OK)
```json
{
  "intent": "string (The identified intent, e.g. Attendance, Marks, Multi-intent)",
  "plan": [
    "string (Step-by-step execution path generated by the planner)"
  ],
  "tool": "string (Selected tool names comma-separated)",
  "response": {
    "tool_name": {}
  },
  "summary": "string (The synthesized natural language summary from the agent)",
  "status": "string (Good, Paid, Warning, Critical, or Pending status)",
  "execution_time": 0.0
}
```

### Error Responses
- **400 Bad Request**: Raised when user query message is empty.
  ```json
  { "detail": "User query message cannot be empty." }
  ```
- **404 Not Found**: Raised when `student_id` does not match active records.
  ```json
  { "detail": "Student ID 'ST999' is invalid or does not exist in ERP records." }
  ```

## 📤 `GET /chat/history` - Retrieve Conversations Log

Pulls chat histories stored in the SQLite database database tables.

### Request Details
- **URL Path:** `/chat/history`
- **Method:** `GET`
- **Query Parameters:**
  - `student_id` *(string, optional)*: Filter transcripts specifically by student.
  - `limit` *(int, default=50)*: Cap returned array size (1 to 100).

### Response Schema (200 OK)
```json
[
  {
    "id": 1,
    "student_id": "ST101",
    "query": "Show my midterm marks and check pending fees.",
    "intent": "Multi-intent",
    "tool_used": "Marks Tool, Fees Tool",
    "response": {
      "summary": "Your midterm average is..."
    },
    "timestamp": "2026-07-03T09:20:30Z",
    "execution_time": 1.2
  }
]
```

## ⚡ cURL Execution Example

To submit a query using your terminal:
```bash
curl -X POST "http://127.0.0.1:8000/chat" \
     -H "Content-Type: application/json" \
     -H "X-Groq-API-Key: gsk_your_key_here" \
     -d '{"student_id": "ST101", "message": "Do I have homework due tomorrow?"}'
```
